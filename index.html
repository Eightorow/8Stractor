How to Obfuscate and Run Your Script

To ensure your script is obfuscated and can be run by others, follow these steps:

  1. Obfuscate Your Script:
    Drag and drop your script file onto the 8Stractor.py script.
    This will generate an obfuscated file with the .py8s extension in the same directory as your original script. For example, if your script is named example.py, the obfuscated file will be example.py8s.
  2. Distribute the Files:
    Send the newly created .py8s file and the Run 8s.py script to your friends.
  3. Running the Obfuscated Script:
    Your friends should run the Run 8s.py script.
    They will be prompted to enter the directory where the obfuscated .py8s file is located. Note: This should be the directory path, not the file name.
    After specifying the directory, they will need to type the file name of the obfuscated file (without the .py8s extension).
    The script will then run the obfuscated file.

Example:

  • Obfuscate:
    Original script: example.py
    Output file: example.py8s
  • Distribute:
    Files to send: example.py8s and Run 8s.py
  • Run:
    Directory: C:\\Users\\username\\Desktop\\
    File Name: example.py8s
